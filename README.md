# ListenLess

ListenLess is an Android accessibility app designed to provide real-time speech-to-text captions for deaf and hard-of-hearing users.

## Features
- Speech-to-text transcription
- Modern dark UI
- Accessibility-focused design
- Import and Notes sections (WIP)

## Tech Stack
- Java
- Android Studio
- Android Speech Recognition API
